# Adding AI assist steering

## Overall steering guidance
Leverage the coding-standards.md and the implementation.md for overall steering that should apply to all or nearly all assets

## Module or aspect-specific steering
Any targeted steering should use an appropriate subfoolder from the .ai/specs

## Local or experimental steering
Gitignore will exclude any files or folders under an .ai/local folder. Create that folder as needed and place any steering not intended for broader inclusion
